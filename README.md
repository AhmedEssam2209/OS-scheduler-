<h1>OS-scheduler-</h1>



<h2>Description</h2>
Designed and implemented a CPU operating system scheduler with different scheduling algorithms using C in  Linux platform. 

Data structures used:
1.	Queue
2.	Priority Queue
3.	Circular Queue

Assumptions:
•	Processes are ordered by arrival time from file.
•	If two processes have the same priority, the arrival time of the process decides which process runs first.
•	If two processes have the same priority and arrival time, the process ID of the process decides which process runs first.
•	Time 0 isn’t taken into account in calculations


<h2>Languages and Utilities Used</h2>

- <b> C </b>
- <b> WSL Linux </b>
